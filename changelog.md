# changelog

## [1.2.0] - 2026-04-04 - 6cef9f5

### Добавлено
- Переход на SQLite для постоянного хранения паролей
- База данных `passwords.db` для хранения записей
- Улучшенная обработка путей к файлам данных

### Изменено
- `PasswordManager` теперь использует SQLite вместо `std::vector`
- Пути к файлам master-key и базы данных исправлены для корректной работы

### Удалено
- Зависимость от in-memory хранения данных

## [1.1.0] - 2026-03-31 - dd6c1a3

### Добавлено
- Модуляризация кода: разделение на отдельные классы и файлы
- Классы: `PasswordManager`, `MasterKeyManager`, `FileManager`, `PasswordGenerator`, `ClipboardManager`
- Поддержка master-key для аутентификации
- Генерация случайных паролей
- Копирование в буфер обмена

### Изменено
- Рефакторинг монолитного кода на модульную архитектуру
- Улучшенная структура проекта с папками `src/`, `include/`

## [1.0.0] - 2026-03-31 - 660434d

### Добавлено
- Базовый менеджер паролей на C++
- Функции: добавление, просмотр, поиск, обновление, удаление записей
- Консольный интерфейс с меню
- Хранение паролей в памяти (std::vector)
