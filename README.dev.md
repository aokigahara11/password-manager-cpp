# password-manager-cpp - руководство для разработчиков

Краткое руководство для разработчиков проекта `password-manager-cpp`.

**Цель**: Создание консольного менеджера паролей с поддержкой SQLite и шифрованием.

**Структура проекта**:

```
password-manager-cpp/
│
├───.gitignore                          # Игнорируемые файлы Git
├───changelog.md                        # История версий
├───LICENSE                             # Лицензионное соглашение
├───README.dev.md                       # Документация для разработчиков
│
├───include/                            # Заголовочные файлы
│   ├───password_manager/               # Модуль менеджера паролей
│   ├───sqlite3/                        # SQLite3 API
│   ├───ui/                             # Интерфейс пользователя
│   └───utils/                          # Вспомогательные утилиты
│
├───releases/                           # Скомпилированные версии
│   ├───1.0.0/                          # Версия 1.0.0
│   ├───1.1.0/                          # Версия 1.1.0
│   ├───1.2.0/                          # Версия 1.2.0
│   ├───1.3.0/                          # Версия 1.3.0
│   └───1.3.1/                          # Версия 1.3.1
│
├───src/                                # Исходный код
│   ├───main.cpp                        # Точка входа в приложение
│   ├───data/                           # Файлы данных (БД, ключи)
│   ├───password_manager/               # Логика менеджера паролей
│   ├───ui/                             # Реализация интерфейса
│   └───utils/                          # Реализация утилит
│
└───third_party/                        # Сторонние библиотеки
    └───sqlite3/                        # SQLite3 (амальгама)
```
